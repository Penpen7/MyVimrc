# Getting Started
- Install Dependencies
- Clone this repository into your home directory.
```
cd ~
git clone https://github.com/Penpen7/dotfiles
```
- Change the directory and run setup.sh
```
cd dotfiles
git submodule update --init --recursive
./setup.sh
```
- Run or rerun zsh
- Enjoy!
